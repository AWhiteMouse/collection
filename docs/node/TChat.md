# TChat

### 项目描述

一个基于Node服务的在线聊天室。

#### 已实现

- [x] 在线聊天
- [x] 统计在线人数
- [x] 加入群聊、退出群聊提醒
- [x] 支持快捷消息发送

#### TODO

- [ ] 优化输入昵称的交互
- [ ] 自动分配头像
- [ ] 引入AI自动回复机器人，增加可交互性
- [ ] 引入音乐播放器，在一个愉快地氛围里聊天
- [ ] ……

### 技术栈

`NodeJs`、`socket.io`、`JQuery`
