# GImage

### 项目描述

图片处理、图片生成工具。

#### 已实现

- [x] 生成指定宽高的图片
- [x] 修改原始图片宽高
- [x] 支持导出`.png`，`.png`，`.jpeg`等三种格式的图片
- [x] 支持水印
- [x] 支持单图下载
- [x] 支持批量多图下载

#### TODO
